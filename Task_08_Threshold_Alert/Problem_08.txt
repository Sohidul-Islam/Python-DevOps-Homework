Task 8: The Threshold Alert (Floats & Strings)

Scenario: Your monitoring tool returns a CPU load of `0.88`.

Goal: Format this float as a percentage string (e.g., `"88%"`) using **f-strings**.

DevOps Value: Creating human-readable alerts for Slack or Email.

Instructions:
1. Start with CPU load as a float: 0.88
2. Convert to percentage (multiply by 100)
3. Format as a string with "%" symbol using f-strings
4. Print the formatted alert message
